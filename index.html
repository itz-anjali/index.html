The Challenge: Information Overload

New students are often overwhelmed with questions, leading to repetitive inquiries for staff and seniors. Our goal was to create a single, reliable source for answers.

"Does the college offer this course?"

"What are the fees for BCA?"

"How do I apply for admission?"

"Who is the HOD of my department?"

"Can I pursue tech without a strong math background?"

"What are the hostel facilities like?"

Under the Hood: How Sofiya Works

Built with modern web technologies, Sofiya combines a simple interface with intelligent features to deliver a seamless user experience.

Frontend Interface

Clean UI built with HTML and styled with Tailwind CSS for a fully responsive experience.

Core Logic

Powered by JavaScript, using a predefined Knowledge Base to provide accurate answers.

Tone Detection

Adapts its conversational style based on the user's selected mood: Funny, Motivational, or Serious.

Web Speech API

Enables voice responses in a female voice, making the interaction more engaging and accessible.

Proactive Guidance

Offers animated, clickable options for broad queries to guide users to the right information quickly.

Mentorship Approach

Designed to communicate like a helpful teacher or friend, fostering a supportive environment.
